## Shot list / recording tips
- **Record the demo first**, in one clean pass, at 1920×1080; then trim to the beats above and lay VO on top.
- **Pre-seed the demo:** run the fusion demo once before recording so the feed has data; set `FORCE_GNN_FAILURE=1` briefly to guarantee the **self-heal toast** fires on cue (then turn it off).
- **Hard-refresh** before recording so the **splash screen** plays (great 2-second opener).
- Keep cursor movements slow and deliberate; pause ~1s on each verdict so viewers can read it.
- **Music:** low, tense synth for 0:00–0:55, lift to hopeful at 1:40.
- **Captions:** burn in subtitles — many judges watch muted first.
- **Backup:** if live site is slow, use a pre-recorded screen clip; never demo live-live on a flaky network.

## Word-count check
VO ≈ 260 words ≈ 1:45 spoken, leaving ~15s of breathing room for demo pauses. If you run long, cut the last sentence of the 0:55 beat.
